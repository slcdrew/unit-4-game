Crystal Collector


You will be given a randomly generated Target Score at the start of the game.

Below, there are four clickable crystal images.

By clicking on a crystal imnage the user adds a specific amount of points to the User Total.

To win the game the User Total must match the Target Score

If the User Total surpasses the Target Score the player losses.

The amount of points each crystal is worth is intentionally hidden from the player.

The Target Score and crystal values are randomized every game.
